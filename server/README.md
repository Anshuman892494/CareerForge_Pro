# ⚙️ CareerForge Pro - Server

This is the backend API for the CareerForge Pro AI Career Suite.

## 🚀 Features
- **AI Career Logic**: Integration with Gemini and Groq for resume optimization.
- **ATS Analysis**: Algorithms for job description matching and scoring.
- **PDF Engine**: Server-side PDF generation using Puppeteer.
- **Data Management**: MongoDB integration for user data and configurations.

## 🛠️ Tech Stack
- Node.js & Express 5
- MongoDB & Mongoose
- Google Generative AI (Gemini)
- Groq Cloud SDK
- Puppeteer

## 🏃 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure Environment:
   Create a `.env` file based on `.env.example`.

3. Start development server:
   ```bash
   npm run dev
   ```

For more information, see the [Main README](../README.md).
