# 🎨 CareerForge Pro - Client

This is the frontend portion of the CareerForge Pro AI Career Suite.

## 🚀 Features in Frontend
- **Real-time Resume Builder**: Dynamic form-to-preview synchronization.
- **AI Integration**: Interface for AI bullet point optimization and cover letter generation.
- **Export System**: Seamless PDF generation and download.
- **Responsive Design**: Optimized for all screen sizes using Tailwind CSS 4.

## 🛠️ Tech Stack
- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide React

## 🏃 Running Locally
Ensure the backend server is running first, then:
```bash
npm install
npm run dev
```

For more information, see the [Main README](../README.md).
